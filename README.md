# ♟️ AI Chess Engine

A functional chess game with an integrated AI opponent. This project explores game logic, state evaluation, and automated decision-making using Python.

## 🌟 Key Features
* **Custom AI Opponent:** Uses algorithmic logic to evaluate the board and choose the best moves.
* **Full Chess Rules:** Implementation of all standard moves, including castling and pawn promotion.
* **Game State Analysis:** Real-time tracking of check, checkmate, and stalemate conditions.

## 🛠 Technology Stack
* **Language:** Python 3.x
* **Core Logic:** Developed with focus on modularity and algorithmic efficiency.

## 📊 AI & Logic
The core of this project is the decision-making engine. It processes the current board state as data and applies a set of rules to simulate strategic thinking. This demonstrates my ability to:
1. Translate complex real-world rules into clean Python code.
2. Build automated systems that react to user input.
3. Manage complex data structures (the chessboard).
